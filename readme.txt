== Description ==
The new booking plugin for wordpress (super beta).
.html is included in the zip package.

== Development Blog ==

[Giuseppe Surace WP development blog](http://www.giuseppesurace.com "Italian Wp designer and plugin developer")

== Installation ==

1. Upload `bookandpay dir` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to settings and edit your booking
